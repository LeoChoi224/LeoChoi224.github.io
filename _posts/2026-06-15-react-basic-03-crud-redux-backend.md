오늘은 지난주에 이어서 React CRUD 예제를 마무리하고, React에서 상태를 관리하는 여러 방식에 대해 배웠다.

단순히 컴포넌트 내부에서 `useState`로 상태를 관리하는 것에서 시작해서, props를 통한 데이터 전달, Flux 패턴, Context API, Redux까지 흐름을 확장했다.

마지막에는 React 프론트엔드와 FastAPI 백엔드를 연결해서 실제 도서 관리 CRUD 기능을 구현해보았다.
